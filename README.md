AI-generated summaries of noteworthy issues fixed across Drupal projects.

Each summary covers what changed, why it matters, and what action (if any) you need to take.

Not every fixed issue appears in the digest. Issues with minor changes are excluded.

## Browse

Individual issue summaries are also available as markdown files in the [`issues/`](https://github.com/dbuytaert/drupal-digests/blob/main/issues/) directory.

## About

Summaries are generated by AI and may contain errors. Each one links back to the original issue on Drupal.org.

Created and maintained by [Dries Buytaert](https://dri.es/). Questions or suggestions? [Open an issue](https://github.com/dbuytaert/drupal-digests/issues).
