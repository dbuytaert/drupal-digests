## Summary

The bundled CKEditor 5 library has been updated to version 47.6.0 across all supported Drupal branches (main, 11.x, 11.3.x, and 10.6.x). Sites using CKEditor 5 as their rich text editor receive whatever bug fixes and improvements the upstream 47.6.0 release includes. No Drupal integration code was changed.

## Impact

No upgrade or configuration changes required.

## Technical details

All vendored JavaScript asset files under `core/assets/vendor/ckeditor5/` were replaced with their 47.6.0 builds. The diff is minified, but one substantive code change is visible in the `basic-styles` bundle: `AttributeCommand.execute()` now passes `{includeEmptyRanges: true}` to `getValidRanges()` and explicitly handles collapsed ranges by resolving the attribute key via `ModelDocumentSelection._getStoreAttributeKey()` before calling `setAttribute`/`removeAttribute`. The copyright year in the license header block of every bundle was also updated from 2025 to 2026. The upstream release notes are available at https://github.com/ckeditor/ckeditor5/releases/tag/v47.6.0.
