## Summary

The Default Admin theme's description text in its `.info.yml` was reworded to remove external links to gin-theme documentation and to match the plain, link-free style used by the Olivero and Claro theme descriptions. Three now-unnecessary `cspell:ignore is-autocompleting` inline comments were also removed from Claro's autocomplete CSS and JavaScript files.

## Impact

Affects front-end developers.

- **Theme change:** Default Admin theme description no longer contains external links to gin-theme documentation; the plain-text description is now consistent with Olivero and Claro.

## Technical details

The `description` and `alt text` fields in `core/themes/admin_theme/admin_theme.info.yml` were updated. The old description contained anchor tags pointing to contributed gin-theme documentation; the new description is a self-contained sentence ("A clean, accessible and powerful Drupal administration theme with a strong focus on improving the editorial experience."), and the alt text was corrected from "a Drupal administration theme" phrasing to "the Drupal administration theme". The three removed `cspell:ignore is-autocompleting` comments (two in CSS/SCSS files and one in JavaScript) were in Claro's autocomplete component; the word `is-autocompleting` is already present in Claro's own dictionary or word list, making the per-file suppressions redundant.
