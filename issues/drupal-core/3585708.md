## Summary

The Drupal admin theme now preloads the Inter font. By declaring the font with `preload: true`, the browser fetches it early in the page load sequence, reducing the flash of unstyled text (FOUT) that can appear in the administration UI before the font arrives.

## Impact

No upgrade or configuration changes required.

## Technical details

The change adds a `fonts` key to the `global-styling` library definition in the admin theme's `.libraries.yml` file. The single entry, `font/inter.woff2: {preload: true}`, instructs the libraries API to emit an HTML `<link rel="preload">` tag for the Inter font. This is a direct application of the font-preloading support introduced in [#3366561](https://www.drupal.org/node/3366561), which added a first-class `fonts` key to the libraries API alongside the existing `css` and `js` keys. Inter is the only font used by the admin theme, so this single entry covers the full font set.
