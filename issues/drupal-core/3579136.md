## Summary

This is a test-code-only change with no effect on Drupal sites at runtime. It updates the Menu library's unit tests to avoid notices and deprecation warnings that PHPUnit 12.5.0 will emit when mock objects are used without any expectations configured on them. The fix has broad reach because `LocalTaskIntegrationTestBase`, the base class extended by many core modules' local-task integration tests, was among the classes updated.

## Impact

No upgrade or configuration changes required.

## Technical details

PHPUnit 12.5.0 introduces three related changes: mock objects without expectations produce a notice (stubs should be used instead), `$this->any()` is deprecated (it was equivalent to no expectation), and `with()` on stubs is deprecated. The nine test files in `core/tests/Drupal/Tests/Core/Menu/` were triggering all three. The fix applies a consistent pattern across `ContextualLinkDefaultTest`, `ContextualLinkManagerTest`, `DefaultMenuLinkTreeManipulatorsTest`, `LocalActionDefaultTest`, `LocalActionManagerTest`, `LocalTaskDefaultTest`, `LocalTaskIntegrationTestBase`, `LocalTaskManagerTest`, and `MenuActiveTrailTest`: shared properties that were created with `createMock()` in `setUp()` but only needed as passive doubles are switched to `createStub()`. Where a specific test method does need to set expectations (`expects($this->once())`, `expects($this->never())`, etc.), the stub is replaced by a local `createMock()` call within that test, often via a new helper such as `setUpMockCacheBackend()`, `setUpMockDiscovery()`, or `setUpMockAccessManager()`. These helpers use `ReflectionProperty` to inject the mock directly into the object under test after construction. All calls to `$this->any()` are removed; `->method(...)` is called directly on the stub or mock without a matcher. The `LocalTaskIntegrationTestBase` change is especially impactful because dozens of core modules extend it: the `$argumentResolver`, `$accessManager`, `$route_provider`, `$module_handler`, `$factory`, and `$cacheBackend` passed into the manager there are all converted from mocks to stubs. Property-level PHPDoc comments in several classes update the union type from `MockObject` to `Stub` to reflect the new default. Thirty PHPStan baseline suppressions for the `any()` call in these files are removed from `core/.phpstan-baseline.php`.
