## Summary

This is a purely internal change with no user-facing effect. It fine-tunes GitLab CI pipeline performance by consolidating the slowest functional JavaScript tests, tagging additional slow test classes so the pipeline can schedule them efficiently, reordering jobs to run the slowest first, and adjusting CPU and concurrency settings for the Build and Kernel test jobs.

## Impact

No upgrade or configuration changes required.

## Technical details

Follow-up to [#3577234](https://www.drupal.org/node/3577234) targeting three areas in the CI pipeline configuration and test suite.

**Job reordering and resource tuning:** The `PHPUnit Build` job was moved earlier in `.gitlab-ci.yml` so the slowest jobs claim runners first, letting faster jobs fill in behind them. Its `KUBERNETES_CPU_REQUEST` was raised from `"8"` to `"12"` and `CONCURRENCY` from `"5"` to `"6"`. The `PHPUnit Kernel` job received a CPU increase from `"8"` to `"12"` while its `CONCURRENCY` was reduced from `"24"` to `"20"` to avoid overloading runners.

**`BigPipeRegressionTest` consolidation:** This was identified as the single slowest functional JS test. Previously the class had four independent public test methods (`testMultipleClosingBodies_2678662`, `testMessages_2712935`, `testPlaceholderHtmlEdgeCases`, `testMultipleOccurrences`), each installing modules and logging in separately. They were merged into a single public `testPlaceholderHtmlEdgeCases` method that calls all sub-methods as `protected do*` helpers after one shared login. The `render_placeholder_message_test` module was added to the class-level `$modules` array and `$defaultTheme` was set to `big_pipe_test_theme` directly, removing the per-test `setUp()` overhead. `doTestBigPipeLargeContent` was also changed from `public` to `protected` for consistency.

**`#[Group('#slow')]` tagging:** Ten test classes that were observed running past the end of their CI jobs were annotated with `#[Group('#slow')]`: `EntityLinkSuggestionsTest`, `ContentTranslationLanguageActivationTest`, `BooleanFormatterSettingsTest`, `MediaStandardProfileTest`, `WidgetUploadTest`, `UnknownPathExcluderTest`, `PhpExtensionsValidatorTest`, `UpdatePathTestBaseFilledTest`, `StandardTest`, and `InstallerTranslationMultipleLanguageKeepEnglishTest`.
