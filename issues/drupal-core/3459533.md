## Summary

The `isConfigurable()` helper method on `PluginBase` has been deprecated in Drupal 11.1 and will be removed in Drupal 12. The method was introduced during the earlier migration away from `ConfigurablePluginInterface` and has always been a thin wrapper around an `instanceof ConfigurableInterface` check. Now that migration is complete, the method serves no purpose beyond what a direct `instanceof` check provides. Any code that calls `$plugin->isConfigurable()` must be updated to use `$plugin instanceof ConfigurableInterface` directly.

## Impact

Affects module maintainers.

- **Deprecation:** `PluginBase::isConfigurable()` deprecated in 11.1.0, removed in 12.0.0. Use `instanceof ConfigurableInterface` instead.

## Technical details

The method `PluginBase::isConfigurable()` was added in [#2946122](https://www.drupal.org/node/2946122) to ease the transition from `ConfigurablePluginInterface` to `ConfigurableInterface`. A parallel static helper, `PluginHelper::isConfigurable()`, was introduced at the same time to avoid problems with mocks in tests (because `isConfigurable()` is not declared on any interface). That static helper was deprecated in 10.1 via [#3105685](https://www.drupal.org/node/3105685) and removed in Drupal 11. Now that `ConfigurablePluginInterface` is also gone, the instance method on `PluginBase` is equally redundant. The commit adds a `@deprecated` docblock and a `@trigger_error(E_USER_DEPRECATED)` call inside `PluginBase::isConfigurable()` in `core/lib/Drupal/Component/Plugin/PluginBase.php`. The deprecation message directs callers to replace the call with a direct `instanceof \Drupal\Component\Plugin\ConfigurableInterface` check. No BC shim is needed because the method body is unchanged; only the deprecation notice is new. The method will be removed in Drupal 12.0.0. The corresponding change record is at https://www.drupal.org/node/3198285.

## Upgrade

Replace any call to `$plugin->isConfigurable()` with a direct `instanceof` check:

```php
// Before (deprecated in 11.1, removed in 12.0):
if ($plugin->isConfigurable()) { ... }

// After:
use Drupal\Component\Plugin\ConfigurableInterface;

if ($plugin instanceof ConfigurableInterface) { ... }
```
