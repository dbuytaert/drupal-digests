## Summary

The `_filter_tips()` function is deprecated. This function retrieves filter tips for text formats and was only used once in core, in the theme layer. The logic has been moved to a protected method in the `FilterThemeHooks` class as part of the effort to eliminate core `.module` files. This is an internal refactoring with no user-facing changes. Sites using this function in custom code will need to implement their own logic to retrieve filter tips.

## Impact

Affects module maintainers.

- **Deprecation:** `_filter_tips()` deprecated with no public replacement

## Technical details

The `_filter_tips()` function in `filter.module` retrieves an array of filter tips for one or all text formats accessible to the current user. It was only called from `FilterThemeHooks::preprocessFilterGuidelines()`.

The logic is moved to a new protected method `FilterThemeHooks::getFilterTips()`. The method is functionally identical to the original function: it calls `filter_formats()` with the current user, optionally filters to a single format ID, then iterates over each format's filters to build an array of tips keyed by format label and filter name. The only difference is that it uses an injected `AccountInterface` dependency instead of calling `\Drupal::currentUser()`.

`FilterThemeHooks` now has a constructor that accepts `AccountInterface $currentUser` as a dependency. The preprocessing hook calls `$this->getFilterTips()` instead of the deprecated function.

The original function remains in place with a deprecation trigger and is marked as having no replacement, since the new method is protected and intended only for internal use.

## Upgrade

If custom code calls `_filter_tips()`, implement the logic directly by iterating over filter formats and their filters:

```php
$formats = filter_formats($currentUser);
$tips = [];
foreach ($formats as $format) {
  foreach ($format->filters() as $name => $filter) {
    if ($filter->status) {
      $tip = $filter->tips();
      if (isset($tip)) {
        $tips[$format->label()][$name] = [
          'tip' => ['#markup' => $tip],
          'id' => $name,
        ];
      }
    }
  }
}
```
