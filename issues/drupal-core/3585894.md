## Summary

Drupal core now includes an `AGENTS.md` file and a `CLAUDE.md` file that coding agents read automatically when working in the repository. The files set "harm reduction" rules aimed at limiting the negative impact of LLM-generated contributions on human reviewers: discouraging LLM use for prose (issue comments, MR posts, commit messages), requiring a disclosure line on any AI-assisted prose that does get published, capping writing length and banning editorial boilerplate, and directing agents to run Drupal's code-check script and PHPUnit before submitting. This is a three-month trial on `main` only; a follow-up issue ([#3606814](https://www.drupal.org/node/3606814)) tracks evaluation of the effects.

## Technical details

The commit adds two files to the repository root. `AGENTS.md` contains five numbered operator-level sections: **Disclosure** (append "Generated with the help of an LLM." verbatim, no model or vendor name, because smaller local models hallucinate their own names), **Writing Style** (≤180 words, ≤25-word sentences, simplified English, no editorial adjectives like `robust` or `seamless`, no formula phrasing, no restating prior comments), **Code Style** (Drupal coding standards; run `./core/scripts/dev/commit-code-check.sh`), **Tests** (run `./vendor/bin/phpunit -c core <path>` for the matching `tests/` directory), and **Git Commits** (same word and sentence caps, no LLM-credit trailers such as `Co-Authored-By:` or `Assisted-by:`). A "Discouraged uses" preamble precedes the numbered sections: it explicitly discourages using LLMs for any drupal.org prose or for evaluating contributors; if the operator asks anyway, the agent must tell them to rewrite the output in their own words before publishing.

`CLAUDE.md` contains a single line, `@AGENTS.md`, which is the Claude Code reference syntax that causes it to import and apply `AGENTS.md` automatically. This file was added after alexpott noted that Claude Code does not read `AGENTS.md` by default; webchick supplied the one-line `@AGENTS.md` import syntax from the Claude Code documentation.
