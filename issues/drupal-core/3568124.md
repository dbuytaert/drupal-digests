## Summary

Drupal continues modernizing the Media module by moving the last two procedural functions from `media.module` into the `MediaHooks` class. This completes the transition to object-oriented hook implementations. The deprecated functions were already internal or used only as form callbacks, so most sites will not be affected. This change is part of a broader effort to clean up legacy procedural code across Drupal core.

## Impact

Affects module maintainers.

- **Deprecation:** `_media_get_add_url()` and `media_filter_format_edit_form_validate()` deprecated in `media.module`, removed in Drupal 12.

## Technical details

Two functions remained in `media.module`: `_media_get_add_url()` and `media_filter_format_edit_form_validate()`. Both have been deprecated and their code moved to `MediaHooks`.

The `_media_get_add_url()` function was already marked `@internal` and had no known contrib usage. It has been moved to a protected `MediaHooks::addUrl()` method and deprecated without a public replacement. The function generates URLs for adding media based on allowed bundles and access permissions.

The `media_filter_format_edit_form_validate()` function validated that filter configuration is compatible with media embed requirements. It has been moved to the public method `MediaHooks::formatEditFormValidate()`. Form alter hooks now reference the new method using `self::class . ':formatEditFormValidate'` instead of the function name string.

Both deprecated functions trigger an `E_USER_DEPRECATED` error and forward to their new implementations, maintaining backward compatibility. Test classes were updated to use `CoversMethod` instead of `CoversFunction` attributes. The deprecation targets Drupal 12.0.0 for removal, which is allowed for internal functions and form callbacks.

## Upgrade

If you call `_media_get_add_url()` directly, there is no public replacement. This function was internal and should not have been used. If you need similar functionality, implement your own logic based on media bundle access.

If you reference `media_filter_format_edit_form_validate` in custom code, replace it with the new method:

```php
// Before:
$form['#validate'][] = 'media_filter_format_edit_form_validate';

// After:
$form['#validate'][] = \Drupal\media\Hook\MediaHooks::class . ':formatEditFormValidate';
```

See the change record at [#3566774](https://www.drupal.org/node/3566774) for full details.
