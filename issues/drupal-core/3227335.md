## Summary

Drupal 11.1 deprecated 23 HTML-inspection assertion methods in `AssertContentTrait` that were left over from the Simpletest era and are no longer used anywhere in core. They are scheduled for removal in Drupal 12.0.0. Contributed and custom modules that extend kernel tests and call these methods will start receiving deprecation warnings and must remove the calls before upgrading to Drupal 12.

## Impact

Affects module maintainers.

- **Deprecation:** 23 XPath/form-inspection assertion methods in `AssertContentTrait` deprecated in drupal:11.1.0 with no replacement; removed in drupal:12.0.0

## Technical details

`\Drupal\KernelTests\AssertContentTrait` (at `core/tests/Drupal/KernelTests/AssertContentTrait.php`) is a mixin that provides XPath-based HTML-parsing assertion helpers. These helpers were originally written for Simpletest. As Drupal migrated to PHPUnit-based kernel tests and moved HTML-heavy assertions into functional tests backed by `WebAssert`, the XPath/form-inspection helpers in this trait became unused in core.

The commit adds a `@trigger_error(__METHOD__ . '() is deprecated in drupal:11.1.0 and is removed from drupal:12.0.0. There is no replacement. See https://www.drupal.org/node/3476110', E_USER_DEPRECATED)` call at the top of each of the 23 targeted methods, and updates their PHPDoc blocks with a `@deprecated in drupal:11.1.0` tag. No replacement API is provided; the guidance is that tests requiring this kind of assertion should use `WebAssert` in a functional test context.

The 23 deprecated methods are: `assertNoLinkByHref`, `assertNoLinkByHrefInMainRegion`, `assertNoEscaped`, `assertUniqueText`, `assertNoUniqueText`, `assertUniqueTextHelper`, `assertTextPattern`, `assertNoTitle`, `assertNoFieldByXPath`, `assertNoFieldByName`, `assertFieldById`, `assertNoFieldById`, `assertFieldChecked`, `assertNoFieldChecked`, `assertOption`, `assertOptionByText`, `assertOptionWithDrupalSelector`, `assertNoOption`, `assertOptionSelected`, `assertOptionSelectedWithDrupalSelector`, `assertNoOptionSelected`, `assertNoField`, and `assertNoDuplicateIds`.

## Upgrade

Remove all calls to the deprecated methods. There is no drop-in replacement. If the assertion logic is still needed, rewrite the test as a functional test and use `WebAssert` (e.g. `$this->assertSession()->elementExists(...)`, `$this->assertSession()->optionExists(...)`, etc.).

Example of a method that is now deprecated:
```php
// Before (deprecated in 11.1, removed in 12.0):
$this->assertNoLinkByHref('/some/path');
$this->assertFieldChecked('edit-my-checkbox');
$this->assertOption('edit-my-select', 'value');
$this->assertNoDuplicateIds();
```

If these checks are required, move the test to extend `BrowserTestBase` (or `WebDriverTestBase`) and use `WebAssert`:
```php
// After (functional test with WebAssert):
$this->assertSession()->linkByHrefNotExists('/some/path');
$this->assertSession()->checkboxChecked('edit-my-checkbox');
$this->assertSession()->optionExists('edit-my-select', 'value');
```
