## Summary

Drupal core documentation and comments have been updated to remove outdated references to "Drupal 8" where the version number is no longer necessary. The codebase now uses generic "Drupal" terminology for concepts that apply to all modern versions, making documentation more accurate and less confusing for developers. References to "Drupal 8" remain only where historically specific, such as in migration paths or when explaining legacy decisions. This cleanup improves documentation clarity without changing any functionality.
