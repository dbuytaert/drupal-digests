## Summary

Drupal now requires PostgreSQL 18 as the minimum supported version for sites using PostgreSQL. This aligns with the platform requirements set for Drupal 12. Sites running older versions of PostgreSQL will need to upgrade their database server before upgrading to this version of Drupal.

## Impact

Affects site owners.

- **Requirements:** Minimum PostgreSQL version raised from 16 to 18

## Technical details

The `PGSQL_MINIMUM_VERSION` constant in the PostgreSQL driver was updated from 16 to 18. The GitLab CI configuration was also updated to test against PostgreSQL 18 on commits instead of PostgreSQL 16. The merge request job matrix was simplified to only test PostgreSQL 18, while the separate daily PostgreSQL 18 job was removed since it became redundant with the commit job. This change follows the platform requirements established in [#3534144](https://www.drupal.org/node/3534144) for Drupal 12.
