## Summary

`CssOptimizer::loadFile()` has been changed from `public` to `protected`. The method was only ever made public so that the `color` module could call it directly; since `color` module is being removed from core (see [#3270899](https://www.drupal.org/node/3270899)), there is no remaining reason for the public visibility. Any contrib or custom code that calls `CssOptimizer::loadFile()` directly will break on Drupal main/12.x.

## Impact

Affects module maintainers.

- **API change:** `CssOptimizer::loadFile()` visibility reduced from `public` to `protected`; direct callers outside the class hierarchy will get a fatal error.

## Technical details

The sole justification for `public` visibility was documented in a now-removed doc comment: "the only reason this method is public is so color.module can call it; it is not on the `AssetOptimizerInterface`, so any future refactoring can make it protected." With the color module removal tracked in [#3270899](https://www.drupal.org/node/3270899), that justification is gone. The fix in `CssOptimizer.php` is a one-line visibility change from `public` to `protected` on `loadFile()`, with the explanatory note stripped from the docblock. The method has never been part of `AssetOptimizerInterface` and `CssOptimizer` carries no `@internal` annotation, so callers outside the class hierarchy (contrib modules calling `$optimizer->loadFile(...)`) are directly affected.

## Upgrade

Any contrib or custom code calling `CssOptimizer::loadFile()` directly must be updated. Since `loadFile()` is not on `AssetOptimizerInterface`, there is no interface-compatible replacement. Callers should instead work through the public API exposed by `AssetOptimizerInterface`, or subclass `CssOptimizer` and call the method from within the subclass.

```php
// Before (no longer works on Drupal 12.x):
/** @var \Drupal\Core\Asset\CssOptimizer $optimizer */
$contents = $optimizer->loadFile($file, $optimize, $reset_base_path);

// After: extend CssOptimizer and call from within the subclass:
class MyCssOptimizer extends CssOptimizer {
  public function myMethod(string $file): string {
    return $this->loadFile($file);
  }
}
```
