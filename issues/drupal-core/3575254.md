## Technical details

This issue removes deprecations from nine core modules as part of preparing for Drupal 12. The changes include removing deprecated procedural hooks that were converted to OOP (like `template_preprocess_block()` moved to `BlockHooks::preprocessBlock()`), eliminating obsolete helper functions (such as `_ckeditor5_theme_css()` and locale translation utility functions), and removing unused test modules like `ckeditor5_icon_deprecation_test` and `field_ui_test_deprecated`.

The issue also removes the filter tips pages and their routes (`filter.tips_all` and `filter.tips`), along with `FilterController::filterTips()`. Deprecated batch operation callbacks in the locale module were removed, and various library file movement declarations were cleaned up. The changes update dozens of template file comments to reference the correct OOP hook implementations instead of removed procedural functions.

Two noteworthy finds during review: a test module that underwent OOP conversion in 2024 but was never actually used by any tests, highlighting the maintenance cost of dead test code. The scope was intentionally limited to keep the merge request reviewable, with additional deprecations left for a future part 3.
