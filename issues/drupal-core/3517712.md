## Summary

The `field_layout` module, which provides display control for field groups via the Field Layout system, has been fully removed from Drupal core and migrated to a standalone contributed project at drupal.org/project/field_layout. Sites that rely on Field Layout will need to install and enable the contributed module before upgrading to the version of Drupal core from which it is removed.

## Impact

Affects module maintainers.

- **Deprecation:** `field_layout` core module deprecated and removed; replaced by the contributed drupal/field_layout package (2.0.0 stable release)

## Technical details

This is a meta tracking issue for the full deprecation and removal lifecycle of the `field_layout` core module, following the [core module removal process](https://www.drupal.org/about/core/policies/core-change-policies/module-or-theme-removal-process). The removal was policy-approved in [#3007167](https://www.drupal.org/node/3007167).

The tracked work included:

- Moving non-migration tests and Help Topics to the contributed extension (both completed as sub-tasks listed in the issue body).
- Searching core for remaining cross-module references: `quietone` identified references inside `layout_builder.install` and `LayoutBuilderFieldLayoutCompatibilityTest`; these were resolved in [#3541507](https://www.drupal.org/node/3541507), which moved `field_layout` display update logic from `layout_builder` to the `field_layout` module itself.
- Deprecating the core extension in Drupal 11 core via [#3517719](https://www.drupal.org/node/3517719).
- Establishing a stable contrib release: `andypost` took over as maintainer of the contrib project (after `bnjmnm`, who originally created the drupal.org/project/field_layout project, no longer had bandwidth) and published the `2.0.0` release.
- Removing special composer/packaging treatment for the module via [#3567857](https://www.drupal.org/node/3567857) in the project_composer queue.

## Upgrade

Install the contributed `field_layout` module before upgrading to the Drupal core version that removes it:

```
composer require drupal/field_layout
drush en field_layout
```

The contrib module at https://www.drupal.org/project/field_layout provides the same functionality as the removed core module.
