## Summary

CKEditor 5 has been updated from 47.6.0 to 48.0.1 in Drupal core's `main` branch (Drupal 12 only; Drupal 11 stays on the CKEditor 5 v47 LTS line per an agreement with CKSource). The new version resolves long-standing HTML table problems: the editor no longer collapses multiple `<tbody>` sections or strips `<thead>`, which had impeded screen-reader accessibility. Support for `<tfoot>` and correct `scope` attributes on header cells is also included. This upgrade was blocked until the switch away from the DLL installation format landed in [#3527914](https://www.drupal.org/node/3527914).

## Impact

No upgrade or configuration changes required.

## Technical details

The version strings in `core/assets/vendor/ckeditor5/ckeditor5.libraries.yml` were bumped from `47.6.0` to `48.0.1` across every sub-library entry (the per-plugin libraries such as `ckeditor5.table` remain present but are already deprecated for removal in 13.0.0). The npm devDependency in `core/package.json` was updated from the `~47.6.0` range to `^48.0.1`. CKEditor 5 v48 dropped the DLL build format entirely, so [#3527914](https://www.drupal.org/node/3527914) (switching to the UMD installation method) was a hard prerequisite for this upgrade.

The `core/scripts/js/jsdoc-generate.js` script was also fixed for compatibility with v48's distribution layout: the glob was changed from `./ckeditor5*/src/**/*.+(js|jsdoc)` to `./ckeditor5*/dist/**/*.d.ts` (TypeScript declaration files), and the export-matching regex was extended to recognise `declare`, `interface`, `const`, `type`, and `enum` in addition to `class` and `function`. This was prompted by a review comment from godotislate noting the script produced an empty output against the v48 package structure; fixing it also allowed closing [#3551053](https://www.drupal.org/node/3551053) as "won't fix".

The `AssetAggregationAcrossPagesTest` byte thresholds were updated to reflect the slightly smaller bundle produced by v48: `ScriptBytes` dropped from 3,222,537 to 3,206,361 and `StylesheetBytes` from 1,386,351 to 1,286,838.
