## Summary

The Contact module is being removed from Drupal core and moved to a standalone contributed project. This meta issue tracks the full deprecation checklist: stripping Contact from the Standard and Umami installation profiles, removing the Contact recipe, clearing references to Contact from tests across config_translation, user, views, ajax_test, and other subsystems, deprecating the module in core ([#3520470](https://www.drupal.org/node/3520470)), and finally removing it from core entirely ([#3573484](https://www.drupal.org/node/3573484)). The contrib package drupal/contact 1.0.0 is available as a drop-in replacement for sites that depend on the module.

## Impact

No upgrade or configuration changes required.

## Upgrade

Sites using the core Contact module should add the contributed package as a dependency before upgrading to the Drupal version where the module is removed:

```bash
composer require drupal/contact
```

The contrib module (1.0.0) is a direct replacement and requires no configuration changes.
