## Summary

The `getDrupalRoot()` method on Drupal's base test trait is now deprecated. Test classes that extend Drupal's test infrastructure and call `$this->getDrupalRoot()` or `static::getDrupalRoot()` will trigger a deprecation notice in Drupal 12 and must be updated before Drupal 13, where the method is removed. The direct replacement is the `$this->root` property, which is already available on all Drupal base test classes.

## Impact

Affects module maintainers.

- **Deprecation:** `DrupalTestCaseTrait::getDrupalRoot()` deprecated in drupal:12.0.0 and removed in drupal:13.0.0; use `$this->root` directly in test classes

## Technical details

Following [#3573954](https://www.drupal.org/node/3573954), which consolidated common test-class logic into `DrupalTestCaseTrait`, the `getDrupalRoot()` static method on that trait was left as a now-redundant wrapper. The diff adds a `@trigger_error(E_USER_DEPRECATED)` call to `DrupalTestCaseTrait::getDrupalRoot()` with a deprecation targeting drupal:12.0.0 and removal in drupal:13.0.0, pointing to the change record at https://www.drupal.org/node/3574112.

The `$root` property hook in `DrupalTestCaseTrait` previously lazy-initialized itself by calling `static::getDrupalRoot()`. That call is replaced with the same inline `dirname(substr(__DIR__, 0, -strlen(__NAMESPACE__)), 2)` path expression, so the property no longer triggers the deprecation itself. All in-tree test callers of `getDrupalRoot()` are updated to reference `$this->root` directly.

## Upgrade

Any test class that calls `getDrupalRoot()` should be updated:

```php
// Before (deprecated in drupal:12.0.0, removed in drupal:13.0.0)
$path = $this->getDrupalRoot() . '/core/tests/fixtures/...';
$path = static::getDrupalRoot() . '/core/tests/fixtures/...';

// After
$path = $this->root . '/core/tests/fixtures/...';
```

For `static` contexts such as `@dataProvider` methods where `$this->root` is unavailable, move the root-dependent path construction into the test method body itself and use a placeholder in the provider array, replacing it at test time as shown in one of the in-tree changes in this commit.
