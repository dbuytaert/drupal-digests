## Summary

Three test-infrastructure symbols that were originally deprecated as of Drupal 12.0.0 are now correctly deprecated as of Drupal 11.4.0. The symbols are `TestRequirementsTrait` (the whole trait), `TestRequirementsTrait::getDrupalRoot()`, and `BrowserTestBase::setUpAppRoot()`. A matching deprecation exists on `DrupalTestCaseTrait::getDrupalRoot()`. All are removed in Drupal 13.0.0. If your contributed or custom modules contain test classes that use any of these, you will start seeing deprecation notices on Drupal 11.4 and should migrate to `$this->root` directly.

## Impact

Affects module maintainers.

- **Deprecation:** TestRequirementsTrait, TestRequirementsTrait::getDrupalRoot(), DrupalTestCaseTrait::getDrupalRoot(), and BrowserTestBase::setUpAppRoot() are deprecated from drupal:11.4.0 (previously stated as 12.0.0) and removed in drupal:13.0.0; use $this->root directly

## Technical details

The original work in [#3573954](https://www.drupal.org/node/3573954) and [#3589047](https://www.drupal.org/node/3589047) landed on 12.x using a PHP 8.4 property hook on `DrupalTestCaseTrait::$root`, which meant the changes could not be backported to Drupal 11 (which still supports PHP 8.3). Issue [#3590425](https://www.drupal.org/node/3590425) replaced that hook with a compatible initialization, unblocking the backport. This issue then backports the full changeset to `11.x` and, on `main`, corrects the three deprecation `@trigger_error` strings and docblock annotations from `drupal:12.0.0` to `drupal:11.4.0` in `core/tests/Drupal/Tests/TestRequirementsTrait.php`, `core/tests/Drupal/Tests/DrupalTestCaseTrait.php`, and `core/tests/Drupal/Tests/BrowserTestBase.php`. The removal target remains `drupal:13.0.0` in all cases. The associated change record is [#3574112](https://www.drupal.org/node/3574112).

## Upgrade

Replace any usage of the deprecated symbols in your test classes:

```php
// Before (deprecated as of drupal:11.4.0)
use \Drupal\Tests\TestRequirementsTrait;
$root = self::getDrupalRoot();          // via TestRequirementsTrait or DrupalTestCaseTrait
$this->setUpAppRoot();                  // via BrowserTestBase

// After
$root = $this->root;                    // access the $root property directly
```
