## Summary

The two procedural cookie helpers `user_cookie_save()` and `user_cookie_delete()` are deprecated as of Drupal 11.4 and will be removed in Drupal 13.0. There is no direct replacement: callers should set or clear cookies directly on the Symfony `Response` object. The single core call site in `SystemController::compactPage()` has already been updated.

## Impact

Affects module maintainers.

- **Deprecation:** `user_cookie_save()` and `user_cookie_delete()` deprecated in 11.4, removed in 13.0; set or clear cookies directly on the Symfony `Response` object instead.

## Technical details

Both functions live in `user.module`. `user_cookie_save()` was used once in core, in `SystemController::compactPage()` to toggle the admin compact-mode cookie. `user_cookie_delete()` had no core callers at all. Both now emit `E_USER_DEPRECATED` via `@trigger_error()` and carry standard deprecation docblocks pointing to the change record at `https://www.drupal.org/node/3581570`.

The core call site in `SystemController::compactPage()` was rewritten to construct a Symfony `Cookie` object directly (setting a one-year expiry via `$this->time->getRequestTime() + 31536000` when `$mode === 'on'`, or calling `$response->headers->clearCookie()` otherwise). To support this, `SystemController` gains a new autowired constructor parameter `readonly protected TimeInterface $time`, with the corresponding `use` imports for `TimeInterface` and `Cookie`.

## Upgrade

Replace `user_cookie_save()` by attaching a `Cookie` to the response headers:

```php
// Before
user_cookie_save(['my_setting' => $value]);

// After (inside a controller that has access to the response)
use Symfony\Component\HttpFoundation\Cookie;

$response->headers->setCookie(
  new Cookie('Drupal.visitor.my_setting', $value, \Drupal::time()->getRequestTime() + 31536000)
);
```

Replace `user_cookie_delete()` by clearing the cookie on the response headers:

```php
// Before
user_cookie_delete('my_setting');

// After
$response->headers->clearCookie('Drupal.visitor.my_setting');
```
