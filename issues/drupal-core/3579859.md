## Summary

This is a test-infrastructure change with no effect on production behavior. Unit tests in the Form library were updated to be compatible with PHPUnit 12.5.0, which deprecates `$this->any()` and will emit notices when a mock object is used without configuring any expectations. Site owners and module developers are not affected.

## Impact

No upgrade or configuration changes required.

## Technical details

PHPUnit 12.5.0 introduces two related changes: `$this->any()` is deprecated, and mock objects that have no expectations configured emit a deprecation notice suggesting they should be replaced with stubs. The fix updates the Form library's unit test classes (`FormTestBase`, `FormBuilderTest`, `FormCacheTest`, `FormValidatorTest`, `FormSubmitterTest`, `FormAjaxSubscriberTest`, `FormAjaxResponseBuilderTest`, `ConfirmFormHelperTest`, `ConfigFormBaseTraitTest`, `FormErrorHandlerTest`, `FormStateDecoratorBaseTest`) by replacing `createMock()` calls with `createStub()` wherever no expectations are set, and removing `->expects($this->any())` from stub-only usage. For the few tests that do assert call counts or arguments, the mock is now created inside the specific test method (not in `setUp()`) to keep the mock/stub distinction clear.

The most notable structural addition is `StubFormSubmitter`, a new concrete subclass of `FormSubmitter` in the test namespace that overrides the protected `batchGet()` method to return an empty array. This replaces the previous pattern of using `getMockBuilder(FormSubmitter::class)->onlyMethods(['batchGet'])->getMock()` in both `FormTestBase::setUp()` and `FormSubmitterTest::getFormSubmitter()`. Two helper methods, `setUpMockCsrfTokenGenerator()` and `setUpMockFormCacheInterface()`, were added to `FormTestBase` to allow specific tests that need expectation-based mocks to swap in a real mock for `$this->csrfToken` or `$this->formCache` using reflection, since those properties are initialized as stubs in `setUp()`. The corresponding PHPStan baseline suppressions for `any()` usage across six test files were also removed.
