# #3501978: Direct links to flush caches, run cron, and such

## Summary

Drupal CMS now includes the `navigation_extra_tools` contributed module by default. This gives all Drupal CMS sites one-click links in the navigation toolbar to flush all caches, run cron, run database updates, and other common administrative operations. Previously, site builders had to navigate to Configuration > Development > Performance to clear caches, or install Admin Toolbar Extra Tools separately to get these shortcuts.

## Impact

Affects module maintainers.

- **New API:** New `navigation_extra_tools` module bundled in the base Drupal CMS recipe, providing toolbar shortcuts to flush caches, run cron, and run updates on all Drupal CMS sites

## Technical details

The `navigation_extra_tools` module (version `^1.3`) was added to the `install` list in `recipes/drupal_cms_site_template_base/recipe.yml` and declared as a Composer dependency in `recipes/drupal_cms_site_template_base/composer.json`. Because `drupal_cms_site_template_base` is the foundational recipe applied to every Drupal CMS installation, these toolbar shortcuts are available on all Drupal CMS sites without any additional setup. The module integrates with core's Navigation module, which Drupal CMS already installs. The feature request was prompted by the observation that Drupal CMS does not assume Admin Toolbar is installed, leaving users without the quick-access admin links they were accustomed to from Admin Toolbar Extra Tools.
