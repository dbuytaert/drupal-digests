## Technical details

This issue consolidated documentation updates for the Drupal CMS 2.0 release across multiple merge requests. The changes reorganized the information architecture by moving files between directories: content from `docs/src/create/` was redistributed to `docs/src/get-started/`, `docs/src/customize/`, and in some cases removed entirely. Files moved to `customize/` include add-ons, modules, email setup, hosting, consent management, and privacy policy documentation. New content in `get-started/` covers site settings and theme customization. Several tutorials were deleted, including workflows, site-settings, and detailed navigation guidance, while others were significantly condensed. The contributor removed verbose "Goal," "Prerequisites," "Next steps," and "Wrap-up" sections from most tutorials to streamline the content. Screenshots were updated throughout to reflect the 2.0 administrative interface, and admin paths were corrected (switching from old-style notation to the newer format). The changes also removed references to features not present in 2.0, such as the Events content type, and updated feature descriptions to match current functionality. Binary image files were deleted or replaced as part of the refresh.
