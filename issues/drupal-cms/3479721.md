## Technical details

The diff modifies the welcome dashboard config YAML (inside the `drupal_cms_content_studio` recipe area) and its feature definition. The layout is restructured from a single `layout_onecol` section to a `layout_twocol_section` (75-25 column split) followed by a second `layout_onecol` section for a content listing.

In the two-column section, the `first` column receives: a renamed "Set up analytics" menu block (previously labeled "Welcome"), an "Add new capabilities" `dashboard_text_block` placeholder, a "Site Analytics" `dashboard_text_block` placeholder (body: "Coming soon..."), and a `dashboard_site_status` block. The `second` column receives a `dashboard_text_block` containing raw HTML with primary-button links to `/node/add/page`, `/node/add/event`, and `/node/add/blog`. A `@todo` comment (referencing [#3481417](https://www.drupal.org/node/3481417)) flags those CTA buttons as needing a better UX approach. The new one-column section adds a `views_block:dashboard_content-block_site_content` block to list recent site content.

The feature definition is also updated to include a `config: import: dashboard: '*'` directive, pulling in all config provided by the `dashboard` module.
