## Summary

The `drupal_cms_seo_basic` recipe now suggests the `drupal_cms_seo_tools` recipe via Composer. When a site builder runs `composer require drupal/drupal_cms_seo_basic`, Composer will surface `drupal/drupal_cms_seo_tools` as an optional next step, making it easier to discover the more advanced SEO capabilities (meta tag management, SEO analysis, sitemaps, and an SEO checklist dashboard). Both the Basic SEO and SEO Tools recipe descriptions were also rewritten to better convey what each recipe actually provides.

## Impact

No upgrade or configuration changes required.

## Technical details

The change adds a `suggest` block to `drupal_cms_seo_basic/composer.json` with `drupal/drupal_cms_seo_tools` as the suggested package. The suggestion text reads: "Add additional SEO tools to help configure and manage meta tags, SEO analysis, sitemaps, with an SEO checklist and dashboard."

Alongside the suggestion, descriptions were updated in both recipes' `composer.json` and `recipe.yml` files. `drupal_cms_seo_basic` changed from "Configures Drupal CMS for basic SEO best practices." to "Configures Drupal for basic SEO best practices." `drupal_cms_seo_tools` changed from "Configures Drupal CMS for Search Engine Optimization best practices." to "Adds SEO tools to help configure and manage meta tags, SEO analysis, sitemaps, with an SEO checklist and dashboard." so that the description and the suggestion text echo each other, giving a consistent signal to anyone reading either.
