# #3591789: Deleting a content entity translation leaves its auto-save snapshot behind; publishing resurrects the deleted translation

- **Project:** Drupal Canvas
- **Type:** Major bug report
- **Status:** Fixed
- **Discussion:** [#3591789](https://git.drupalcode.org/project/canvas/-/work_items/3591789) · 1 contributor · 5 comments

## Summary

On Canvas sites using content translation, deleting a single translation of a page left that translation's unpublished draft (auto-save snapshot) behind in the store. The next time the page's pending changes were published, Canvas silently reapplied the stale snapshot and recreated the deleted translation, undoing the editorial decision without any warning or conflict report. After this fix, deleting a translation also discards its draft immediately, so publish produces the expected result.

## Impact

No upgrade or configuration changes required.

## Technical details

The auto-save store holds one snapshot per langcode. `ApiAutoSaveController::applyAutoSaveTranslationSnapshots()` iterates every stored snapshot for an entity's translation group and overlays each one onto a freshly loaded entity; when it finds a langcode that no longer exists it falls through to `addTranslation()`, which is the resurrection path. Canvas already implemented `hook_entity_delete()` in `AutoSaveHooks::entityDelete()` to discard the entire snapshot group when a whole entity is deleted. The single-translation case was not handled because `hook_entity_translation_delete()` had no implementation. The fix adds `AutoSaveHooks::entityTranslationDelete()`, which calls `AutoSaveManager::delete($translation)` with only the deleted translation object, discarding just that langcode's snapshot while leaving sibling translations' drafts untouched. The fix is covered by `testDeletingTranslationDiscardsItsSnapshot()` in `ContentEntityTranslationPropagationTest`, which asserts that after removing a Spanish translation the EN snapshot survives, and that a subsequent publish call does not restore the ES translation to the entity.
