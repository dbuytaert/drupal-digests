## Summary

Canvas now includes documentation for using the `@tailwindcss/typography` plugin with the `FormattedText` component. This plugin applies elegant typography styles to HTML content from Drupal text fields, making it easier to display formatted content with consistent styling. Site builders can now use the `prose` class to automatically style rich text without writing custom CSS for paragraphs, headings, lists, and other elements.

## Impact

No upgrade or configuration changes required.

## Technical details

The documentation was added to the Code Components packages page (`docs/user/src/content/docs/code-components/packages.mdx`). The new section explains how to enable the `@tailwindcss/typography` plugin by adding `@plugin "@tailwindcss/typography";` to Global CSS, which is now supported following [#3553356](https://www.drupal.org/project/canvas/issues/3553356).

The documentation covers key features of the typography plugin: size modifiers (`prose-sm`, `prose-base`, `prose-lg`, `prose-xl`, `prose-2xl`), dark mode support with `prose-invert`, element-specific modifiers like `prose-headings:underline` and `prose-a:text-blue-600`, the `not-prose` class to opt out of styling, and using `max-w-none` to override the default maximum width. Each feature includes code examples showing how to apply these classes to the `FormattedText` component when rendering body content from Drupal.
