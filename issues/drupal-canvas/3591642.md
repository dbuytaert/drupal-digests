## Summary

The sections listing page in Bootstrap UI Kit now shows a "View" button alongside the existing "Manage" button for each section. Clicking "View" opens the UI Kit preview page scrolled to the relevant section in a new browser tab, letting site builders quickly check a section's appearance without leaving the listing.

## Impact

No upgrade or configuration changes required.

## Technical details

The change is in `SectionsListForm`. Previously the operations column held only the `$manage_link` render element. The patch adds a second link built with `Url::fromRoute('bootstrap_ui_kit.ui_kit', [], ['fragment' => $section_key])`, wrapped in a `Link::fromTextAndUrl()` call and rendered as a button with `target="_blank"` and `rel="noopener noreferrer"`. Both links are now nested inside a `#type => 'container'` element with class `operations`, which replaces the bare `$manage_link` in the table row data array.
