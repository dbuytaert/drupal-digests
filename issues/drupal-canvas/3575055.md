## Technical details

The Canvas documentation site adds a new "Link" section to the props documentation page. This section explains how to use JSON Schema's `format: uri` and `format: uri-reference` properties for link validation in Single Directory Component schemas.

The `uri` format requires absolute URLs with a scheme, like `https://example.com` or `internal:/node/1`. The `uri-reference` format accepts both absolute URLs and relative paths like `/about` or `../page.html`. The documentation includes YAML schema examples for both formats and a Twig usage example showing how to render the link value in a component template.

This is documentation-only and does not change any code behavior. It helps developers choose the appropriate format constraint when authoring component schemas that accept link properties.
