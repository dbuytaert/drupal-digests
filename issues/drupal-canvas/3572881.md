## Summary

The canvas editor now supports smooth diagonal scrolling with trackpad gestures, matching the natural feel of design tools like Figma. Previously, panning with a trackpad was restricted to horizontal or vertical movements only. ## Technical details

The editor frame now uses the `@use-gesture/react` library to handle trackpad and touch gestures more naturally. The previous implementation listened to raw wheel events and processed them in discrete steps, throttling events to 50ms intervals and snapping to predefined zoom levels. This caused jerky diagonal scrolling because horizontal and vertical movements were processed separately.

The new implementation uses `useGesture` with separate handlers for wheel and pinch events. Free-form panning applies both X and Y deltas simultaneously, enabling true diagonal scrolling. Zoom behavior differs by browser: Chrome and Firefox fire wheel events with `ctrlKey=true` during trackpad pinch, which the wheel handler processes with continuous scaling based on delta magnitude. Safari fires native GestureEvents, which the pinch handler processes by writing transforms directly to the DOM during the gesture for smooth 60fps updates, then syncing to Redux when complete.

The zoom sensitivity is tuned differently for mouse wheel versus trackpad pinch gestures. Scaling is now logarithmic (proportional to current scale) rather than stepped, providing natural zoom behavior at all levels. The implementation carefully coordinates between gesture state refs and Redux to avoid conflicts between browser-specific event paths.
