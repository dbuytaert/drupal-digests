## Summary

Canvas now shows a live preview of how a page will appear in Google search results. When editing a page's title, URL alias, or meta description in the SEO settings tab, you can see exactly how those changes will look in search results before saving. The preview updates automatically as you type and respects Google's typical character limits for desktop search results.

## Impact

No upgrade or configuration changes required.

## Technical details

The search result preview component (`SerpPreview`) renders within the SEO settings form group on Canvas pages. It reads form values for title, meta title, path alias, and description using Redux selectors from the form state, then applies Google's desktop SERP character limits (60 for title, 160 for description, 70 for URL) with truncation. The component also resolves meta title tokens like `[current-page:title]`, `[canvas_page:title]`, and `[site:name]` to provide accurate preview text. The preview updates reactively as users type because it subscribes to the form state store. The component is inserted into the form via an inline template that renders a custom `<drupal-canvas-serp-preview>` element, which maps to the React component through Canvas's existing Twig-to-JSX bridge.
