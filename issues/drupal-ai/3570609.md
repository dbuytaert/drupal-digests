## Summary

The AI module now supports object detection, a computer vision capability that identifies objects within images and returns their labels, confidence scores, and bounding box coordinates. This enables new use cases like automated image cropping, visual search, and content analysis. Providers that support object detection (like Huggingface with models such as facebook/detr-resnet-50) can now be used through the standardized AI module interface. The optional AI API Explorer submodule includes a testing interface for experimenting with object detection calls.

## Impact

Affects module maintainers.

- **New API:** New `object_detection` operation type for detecting objects in images with bounding boxes and confidence scores

## Technical details

This feature adds a complete operation type for object detection following the AI module's established patterns. The implementation includes `ObjectDetectionInterface` with an `objectDetection()` method, `ObjectDetectionInput` that wraps an `ImageFile`, and `ObjectDetectionOutput` that returns an array of `ObjectDetectionItem` objects. Each item contains a label (e.g. "person", "car"), an optional confidence score (0.0 to 1.0), and a bounding box array with xmin, ymin, xmax, ymax coordinates in pixels.

The API Explorer plugin `ObjectDetectionGenerator` provides a form-based interface where users can upload images and see detected objects rendered in a table. The implementation handles errors gracefully, including missing models, configuration issues, and empty results. Documentation follows the module's pattern with a dedicated page explaining the interface, showing code examples with the Huggingface provider, and linking to the relevant PHP classes. The provider matrix was updated to indicate Huggingface supports this operation.

## Upgrade

Module developers can now implement object detection by having their AI provider plugin implement `ObjectDetectionInterface` and define the `objectDetection()` method:

```php
use Drupal\ai\OperationType\GenericType\ImageFile;
use Drupal\ai\OperationType\ObjectDetection\ObjectDetectionInput;

$image_binary = file_get_contents('photo.jpg');
$normalized_file = new ImageFile($image_binary, 'image/jpeg', 'photo.jpg');
$input = new ObjectDetectionInput($normalized_file);

$detection_items = \Drupal::service('ai.provider')
  ->createInstance('huggingface')
  ->objectDetection($input, 'facebook/detr-resnet-50', ['my-module'])
  ->getNormalized();

foreach ($detection_items as $item) {
  $label = $item->getLabel();
  $score = $item->getConfidenceScore();
  $box = $item->getBox(); // ['xmin' => 535, 'ymin' => 1332, ...]
}
```

See the [Object Detection documentation](https://git.drupalcode.org/project/ai/-/blob/HEAD/docs/developers/call_object_detection.md) for complete details.
