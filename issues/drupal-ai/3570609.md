## Summary

The AI module now supports object detection, a computer vision capability that analyzes images to identify objects and their locations. This enables new use cases like automatic image cropping based on detected subjects, visual search features, and content analysis. Sites can use object detection to help users find images containing specific items, automatically crop product photos to focus on the main subject, or build accessibility features that describe image contents. The feature works with compatible AI providers that support object detection models.

## Impact

Affects module maintainers.

- **New API:** New `ObjectDetectionInterface::objectDetection()` method for detecting objects in images with labels, confidence scores, and bounding boxes

## Technical details

The object detection operation type follows the established AI module pattern for operation types. The implementation includes `ObjectDetectionInterface` with an `objectDetection()` method that accepts an `ObjectDetectionInput` containing an `ImageFile` and returns `ObjectDetectionOutput` with an array of `ObjectDetectionItem` objects.

Each `ObjectDetectionItem` contains a label (the object type), a confidence score (probability from 0 to 1), and a bounding box array with xmin, ymin, xmax, ymax coordinates locating the object in the image. This normalized format allows any provider implementing the interface to return consistent results.

The implementation includes an API Explorer plugin at `/admin/config/ai/explorers/object_detection_generator` where users can upload images, select a provider and model, and test object detection with generated code examples. Documentation covers the interface, input/output models, and usage examples. The Huggingface provider was updated in parallel to support this operation type with models like facebook/detr-resnet-50.

## Upgrade

Module developers can now implement object detection in their AI providers by implementing `ObjectDetectionInterface` and the `objectDetection()` method. Example usage:

```php
use Drupal\ai\OperationType\GenericType\ImageFile;
use Drupal\ai\OperationType\ObjectDetection\ObjectDetectionInput;

$image_binary = file_get_contents('street_photo.jpg');
$image = new ImageFile($image_binary, 'image/jpeg', 'street_photo.jpg');
$input = new ObjectDetectionInput($image);

$provider = \Drupal::service('ai.provider')->createInstance('huggingface');
$result = $provider->objectDetection($input, 'facebook/detr-resnet-50', ['my-module']);

foreach ($result->getNormalized() as $item) {
  $label = $item->getLabel();
  $confidence = $item->getConfidenceScore();
  $box = $item->getBox(); // ['xmin' => ..., 'ymin' => ..., 'xmax' => ..., 'ymax' => ...]
}
```
