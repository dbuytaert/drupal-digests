## Summary

OpenAI deprecated the Dall-E 3 image model on 12 May 2026. This change updates the AI Provider OpenAI module to replace `dall-e-3` with `gpt-image-1` as the default `text_to_image` model, and revises the available image sizes to match those supported by `gpt-image-1`. Existing sites that had `dall-e-3` set automatically as their default will be migrated to `gpt-image-1` when they run database updates. Sites where an administrator explicitly chose `dall-e-3` are not affected by the update hook.

## Impact

Affects site owners.

- **Database update:** `ai_provider_openai_update_10002` migrates any stored `dall-e-3` default to `gpt-image-1` in `ai.settings`.
- **Config change:** Default `text_to_image` model in `ai.settings` changes from `dall-e-3` to `gpt-image-1`; available size options updated to match gpt-image-1 (removes 256x256 and 512x512, adds portrait and landscape variants).

## Technical details

In `OpenAiProvider`, the hardcoded default for the `text_to_image` operation changes from `'dall-e-3'` to `'gpt-image-1'`. The accompanying configuration schema removes the Dall-E 3 size options (`256x256`, `512x512`) and replaces them with the sizes `gpt-image-1` supports: `1024x1024`, `1024x1536`, `1536x1024`, `1024x1792`, and `1792x1024`, with a new default of `1024x1024`.

A new `ai_provider_openai_update_10002()` hook handles existing installations. It reads `ai.settings` and checks whether `default_providers['text_to_image']['model_id']` is exactly `dall-e-3`. If so, it rewrites that key to `gpt-image-1` and saves the config. Sites that manually selected `dall-e-3` retain their stored value; the hook only replaces the old auto-assigned default.

## Upgrade

Run database updates after installing the new module version:

```
drush updb
```

If your site had `dall-e-3` set as the automatic default, it will be migrated to `gpt-image-1`. If you explicitly configured `dall-e-3` and wish to continue using it while OpenAI still supports the model, no automatic change will be made, but you should plan to select a supported model before OpenAI fully removes `dall-e-3`.
