## Technical details

This is a planning and coordination issue for bundling AI Initiative recipes into Drupal CMS 2.0. The issue tracks several feature areas being developed as individual recipes including AI image alt text generation, image classification, LLM-optimized content, content classification, content review workflows, and image file naming.

Features identified for potential inclusion in the 2.0 release include an improved installer that enables anonymous AI usage when AI recipes are selected, AI translation capabilities, image search, and an AI Dashboard module. The decision was made to mark the AI recipe as experimental for the 2.0 release, reducing pressure to have all features production-ready at launch.

More advanced features requiring the Tool module to exit beta status are planned for later releases, including agents that create or edit content, workspace integration for AI-generated changes, content variations, AI search without hallucinations, and Chrome LLM provider support for on-device models.

The issue references coordination with Drupal CMS Helper module rather than creating a separate AI-specific helper module for shipping updates over time.
