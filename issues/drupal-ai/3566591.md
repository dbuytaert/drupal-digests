## Summary

The AI module now supports extracting telephone numbers using artificial intelligence directly from the telephone field widget. When editing content with a telephone field, editors can click a button to trigger AI analysis that extracts phone numbers from other content (like a title or text field) and automatically fills in the telephone field. This makes it easier to capture and normalize phone numbers from unstructured data without manual copying and formatting.

## Impact

No upgrade or configuration changes required.

## Technical details

This feature adds a new Field Widget Actions plugin for telephone fields. The `Telephone` class in `modules/ai_automators/src/Plugin/FieldWidgetAction/Telephone.php` extends `AutomatorBaseAction` and registers itself for the `telephone_default` widget and `telephone` field type through the `FieldWidgetAction` attribute.

The plugin follows the same pattern as the existing `Email` plugin. It implements `aiAutomatorsAjax()` to handle the AJAX request when the automator button is clicked, extracting the form key and field delta from the triggering element. The `saveFormValues()` method populates the telephone widget's `value` element with the AI-generated result, handling both single items and multi-value fields.

The implementation uses the `llm_telephone` automator type, which site builders configure at the field level to specify the input source (like a title field) and prompt for extracting telephone numbers. When triggered from the widget, the configured automator processes the input and returns a formatted telephone number that gets inserted into the field automatically.
