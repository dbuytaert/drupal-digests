# #3566304: The Edit Cog for the Chatbot is hidden behind the chat window.

## Summary

A user interface bug in the FlowDrop UI Agents module prevented site editors from accessing the edit control for the chatbot component. The configuration button was rendered underneath the chat window instead of above it, making it impossible to click. This issue has been resolved as part of a larger migration to the latest version of the FlowDrop UI library.

## Impact

No upgrade or configuration changes required.

## Technical details

This was a CSS z-index or positioning issue in the FlowDrop UI Agents module where the edit cog (contextual link) for the chatbot component was rendered behind the chat window element. This prevented content editors from accessing the configuration interface for the chatbot.

The fix was incorporated into a broader update to the FlowDrop UI library tracked in [#3572310](https://www.drupal.org/project/flowdrop_ui_agents/issues/3572310). The newer version of the FlowDrop UI corrected the CSS layer stacking so the contextual edit controls appear above the chat interface elements.

No code changes are visible in this issue because the resolution came through the dependency update rather than a direct patch to this specific bug report.
