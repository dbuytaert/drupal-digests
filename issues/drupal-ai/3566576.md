## Summary

The AI module now supports generating chart data directly from content using AI. When editing content with a Chart field, editors can click a "Generate Chart" button to analyze structured data in a text field and automatically create a corresponding chart configuration. This makes it easier to visualize data from AI-analyzed content without manually configuring charts. The feature requires the Charts module and works with the existing Field Widget Actions system to trigger AI Automators from the content edit form.

## Impact

No upgrade or configuration changes required.

## Technical details

This feature adds a new Field Widget Action plugin `Chart` in `ai_automators/src/Plugin/FieldWidgetAction/Chart.php` that integrates with the existing `LLM: Chart From Text` automator. The plugin is only available for `chart_config` field types using the `chart_config_default` widget provided by the Charts module.

The implementation follows the same pattern as other Field Widget Action plugins like `Email` and `FAQ`, extending `AutomatorBaseAction`. The `Chart` plugin handles the complex task of populating the chart field's `data_collector_table` structure, which is a nested form element representing chart series and data points.

The plugin's `saveFormValues()` method extracts the chart configuration from the entity's field data and uses a helper method `populateTableValues()` to dynamically populate form elements. This helper handles edge cases like `TranslatableMarkup` objects and maps stored data to the form structure by matching column and row keys.

The Chart automator base class in `ai_automators/src/PluginBaseClasses/Chart.php` was improved to use `str_getcsv()` instead of simple `explode()` for parsing CSV data. It now automatically detects semicolon or comma delimiters and properly handles quoted values containing special characters. The prompt template was also updated to explicitly request semicolon delimiters and proper quoting, fixing a typo ("paranthesis" to "parentheses" and "withe" to "with").

Comprehensive documentation was added in `docs/modules/ai_automators/examples/chart-field.md` with step-by-step setup instructions, and cross-references were added to related documentation files.
